# 404 — Page not found / Oldal nem található

## English

The page you tried to reach doesn't exist (or moved). A few likely reasons:

- The URL was hand-typed and contains a typo
- The page was renamed during the v1.x cleanup (see [CHANGELOG](CHANGELOG.md))
- You followed an old link from before the public-release
- The repo is undergoing maintenance and some content is temporarily offline
