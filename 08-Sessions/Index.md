# Superintelligent Vault meta-sessions

A vault-rendszer önmagán dolgozó session-jei. Minden session a [[11.11-session-protokoll|11.11 protokoll]] szerint zárul: Summary + Learnings + Next + Propagation log.

Csak az SV-meta session-ök vannak public-on. A teljes session-történet (ügyfél-projektek) privát marad.
