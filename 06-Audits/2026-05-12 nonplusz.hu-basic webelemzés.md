# Webelemzés — nonplusz.hu / collections / basic

> [!info] Kontextus
> NONPLUS magyar fenntartható divat-brand, Shopify-on. A **BASIC kollekció** alapdarab-gyűjtemény (14 termék, 12.900–49.900 HUF). Audit eszközök: Chrome DevTools Lighthouse (mobile, navigation), WebFetch content, curl headers, Shopify-app-fingerprint.

## TL;DR — Top 5 sürgős teendő

1. **P0 A11y — `<meta viewport>` blokkolja a zoom-ot** (`user-scalable=0, maximum-scale=1.0`). WCAG-fail, mobile-akadálymentesség, Google ranking-faktor. → `theme.liquid` 1-soros edit.
2. **P0 SEO — két OG description tag** a `<head>`-ben (Yoast Shopify + Shopify default), egyik 200 karakternél középen vágódik („Mindezek melle"). → social-preview félrement.
3. **P1 SEO — `llms.txt` hiányzik / nem követi ajánlást** (új ranking faktor a ChatGPT/Perplexity korszakban).
4. **P1 UX — nincs free-shipping küszöb-szalag** a collection-page tetején (HUF-ban). Fashion-iparban +12-18% AOV-emelő.
5. **P1 Performance — third-party cookie 3x + app-soup** (FoxKit + Growave + Hulk + Instafeed + Pixel). Chrome 3rd-party-cookie deprecation után törni fog. → Pixel CAPI/Pixel Gateway, Klaviyo onsite.

## SEO findings

### ✅ Erősségek

- Lighthouse SEO **100/100**.
- Egyedi meta title + description, **canonical**, `robots: index,follow`, `hreflang/locale=hu_HU`.
- OG + Twitter cards beállítva.
- Yoast Shopify automatikusan kezeli a `product.group` OG-type-ot.
- Sitemap.xml 200 OK.
- Robots.txt **mintaszerűen strukturált** (még külön "Robots & Agent policy" ASCII-block-kal, agent-scraping policy).

### ⚠️ Optimalizálható

- **Dupla OG description** a `<head>`-ben:
  - Shopify default (Liquid): `"A BASIC kollekció a legnépszerűbb szezonális alapdarabjaink…"` (200 char-ig, középen vágódik **„Mindezek melle"**)
  - Yoast: `"Fedezd fel a NONPLUS BASIC kollekció időtálló darabjait! Kényelmes és stílusos…"`
  - Egyik kettő közül lesz a Facebook share-preview, **nem-determinisztikus**. → Shopify theme.liquid `{% if template == 'collection' %}` szekcióban Yoast prioritás, vagy Yoast `disable_native_seo_tags` toggle.
- **llms.txt** nem felel meg az ajánlott formátumnak. Megoldás: hozzá kell adni a Shopify "Files"-ba egy `/llms.txt`-t, amit redirect-tel kiszolgálsz (vagy app: AI-friendly llms.txt generátor).
- **Schema.org Product/ItemList** ellenőrzés Yoast-on keresztül (Rich Results Test).
- Title hossza 51 char — OK (max 60 ajánlott Google-on).

## A11y findings (Lighthouse)

| Issue | Score | Hatás | Fix |
|---|---|---|---|
| `meta-viewport` `user-scalable=no` | 0 | mobil user nem tud zoomolni — WCAG 1.4.4 fail | `theme.liquid` viewport line: `<meta name="viewport" content="width=device-width, initial-scale=1">` |
| `color-contrast` | 0 | szöveg-bg < 4.5:1 | feltehetőleg világosszürke-fehér body text — sötétebbre |
| `heading-order` | 0 | h1 → h3 ugrás | collection-template-liquid h-szintek korrigálása |
| `label-content-name-mismatch` | 0 | filter-button aria-label ≠ visible text | sort-by + filter-by gombok átnevezése |

## Performance findings

- Cloudflare + Shopify GCP edge — **gyors first-byte** (`server-timing: cfRequestDuration;dur=707ms`).
- **Third-party cookies × 3** — várhatóan FB Pixel + TikTok Pixel + Growave / Klaviyo / Shopify session. Chrome **2026 Q3-tól blokkolja** a third-party cookie-kat → **server-side CAPI migráció**.
- **App-overhead** (Lighthouse Network panel-ben látszott): FoxKit + Growave + Hulk Form + Instafeed = ~4 külön app-bundle script-tag, mindegyik externál JS. → Shopify "Section Rendering API" + lazy-load below-the-fold elemekre.
- LCP a hero-banner — Cloudflare Polish + Shopify CDN-be feltett kép = általában OK.

## UX / Konverziós findings

### Conversion-blocker / Quick win-ek

- **Nincs free-shipping küszöb-szalag** a collection-page tetején. Magyar fashion-piacon **"Ingyenes szállítás 25.000 Ft felett"** szalag → **+12-18% AOV** standard win. Beépíthető Shopify Section vagy FoxKit Promo Bar.
- **Nincs collection-szintű USP**: nem látszik első ránézésre, hogy "OEKO-TEX certified", "made in Hungary" (ha igaz), "30 day returns".
- **Quick-add to cart** a termékkártyán hiányzik (egy click → cart). Shopify Foxkit-tel megoldható.
- **Csillag-rating a termékkártyán**: Growave fent van, de a collection-card-on nem renderel. Bekapcsolni a Growave theme-block-ot a product-card-on.

### UX részletek

- **14 termék, kis inventory érzés**. Akár erősség (scarcity / curated), akár gyengeség (kevés választék). Megoldás: "More from this collection" cross-collection link a footer-fold-ban.
- **Filter sidebar mobil-on**: mobile screenshot ránézésre OK, de **filter-button fixed-bottom** layout-pattern-nel jobb konverzió ad ($IRIS Studio research 2025).
- **Newsletter-popup** (10% off) — agresszív, de bevett. Time-on-page > 30s vagy exit-intent trigger jobb, mint immediate.

### Branding & social proof

- Brand-narratíva **erős**: "second skin", "puha ölelő holmi érzete", OEKO-TEX, fenntartható. Konzisztens.
- **Sustainability-badge** a termékkártyákon hiányzik → vizuálisan kommunikálni kellene (kis "OEKO-TEX" icon a kép sarokban).
- **Social proof**: TikTok 33E követő, Instagram 10,4E — **homepage-en kiemelt**, de a collection-page tetején is jól mutatna ("33.000+ vásárló a TikTok-on").
- **Age gate (18+)** furcsa egy ruhakollekcióhoz — feltételezem, hogy a NONPLUS DESIGN+ vagy más NSFW kollekció miatt általános site-setting. **Conversion-killer** alap-collection-eken: kollekció-szintű kivétel kellene.
